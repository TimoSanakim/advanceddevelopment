Advanced development javascript libraries

Week 1

processing 5/p5
p5 is gecreëerd voor graphic en interactive experiences, gebaseerd op Processing. p5 werkt ook met audio video en externe input.
https://p5js.org/

Mustache
dit is een logic-less template syntax. Er zijn dus geen if statements, else clauses of for loops. In Plaats daarvan zijn er tags.
https://mustache.github.io/

Node
node voert javascript code uit buiten de web browser. zo kun je javaScript gebruiken voor command-line-tools en server-side scripting.
https://nodejs.org/en/

Anime
anime.js is een animatie library. Het werkt met onder andere css, svg en javascript object wat anime.js erg flexibel maakt.
https://animejs.com/

Chart
Zoals de naam al impliceert wordt deze javascript library gebruikt om grafieken te maken. Chart.js biedt veel verschillende soorten grafieken aan. Deze library heeft ook de mogelijkheid voor animaties om de grafieken iets interessanter te maken.
https://www.chartjs.org/

Choreographer
is een simpele javascript library waarmee je css mee kan animeren. Deze animaties kunnen worden getriggerd door bijvoorbeeld je scrollwiel.
https://christinecha.github.io/choreographer-js/

Tween
Tween wordt gebruikt voor tweening en animaties. Het ondersteunt het tweenen van numeric object properties & CSS style properties.
https://createjs.com/tweenjs/

Pixi
Pixi.js is een 2d engine die wordt gebruikt om games en interactieve animaties te maken. Disney is een van de grootste bedrijven die hiervan gebruik maakt. Pixi is ook simpel en snel in gebruik.
https://www.pixijs.com/

Raphaël
Raphaël versimpelt het werken met vector graphics op het web. Grafieken en image crops worden door deze library makkelijker gemaakt.
https://dmitrybaranovskiy.github.io/raphael/

Slick
Met slick kan je een carrousel maken die volledig responsive is. Deze kun je ook uitgebreid aanpassen door er filters in te verwerken of het op autoplay te laten lopen
https://kenwheeler.github.io/slick/
